# 📱 QR Code Studio — Free QR Code Generator

A **100% free** online QR code generator website built with Python/Flask. Create beautiful QR codes for URLs, WiFi, WhatsApp, business cards, restaurant menus, and more.

## ⭐ Star Feature: Image QR Code
Upload your logo or photo → get a branded QR code with your image embedded inside. 3 styles: Center Overlay, Background, Color Blend.

## 🎯 Features
- **12 QR Code Types**: URL, Image, WiFi, WhatsApp, vCard, Restaurant Menu, Payment, Location, Email, App Download, PDF, Text
- **Zero Paid APIs**: Uses qrcode[pil], Pillow, segno — 100% free
- **Mobile-First Design**: Dark theme, responsive, 52px touch targets
- **Bilingual**: English + Urdu (اردو) with RTL support
- **Google AdSense Ready**: 5 ad slots per page, ads.txt, sitemap.xml
- **SEO Optimized**: Unique meta tags per page, JSON-LD structured data, FAQ schema
- **Browser History**: QR codes saved to LocalStorage — no database needed
- **Share**: WhatsApp share, copy link, PNG & PDF download

## 🚀 Quick Start
```bash
pip install -r requirements.txt
cp .env.example .env
python run.py
# Open http://localhost:5050
```

## 📁 Tech Stack
- **Backend**: Python 3.11 / Flask
- **QR Generation**: qrcode[pil], Pillow, segno
- **Frontend**: Vanilla CSS (mobile-first dark theme), JavaScript
- **Fonts**: Inter, Noto Nastaliq Urdu
